## detail 클래스의 메소드 중 paintHabits를 분리할 필요가 있음
- 해빗 객체를 통해 로드한 해빗리스트를 받음
- 반복문으로 하나의 해빗씩 그려줌
- 그려준 해빗을 객체로 묶어 디테일 클래스 내에 배열로 저장
- 로컬스토리지에 세이브
  
- 내가 해빗을 하나 추가하고 싶을 때 문제가 발생함
